Data Collection

Using the US Spending API we made calls to gather the 2017 US Spending contracts for the agencies below:

  • Department of Transportation
  • Department of Agriculture
  • Department of the Interior

The required data was stored in a pandas data frame and cleaned before saving as an sqlite file.

Our Flask API was built with enpoints that queried the database.

Heat Map

We created a heatmap using Leaflet.js which referenced one of our Flask API's endpoints.

The map shows the spend over the US with intensity proportional to the Total dollars obligated in each contract.

AnyChart Dashboard

Using the AnyChart Javascript Library we created a dashboard which called another endpoint from our API to obtain the top 10 contracts (based on dollar amount) for each agency.

The dashboard shows a map with points relative to the place of performace and shows the contract's proportions of department spend and total spend when the user clicks a given point.
